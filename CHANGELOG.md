# Changelog

All notable changes to this plugin should be documented in this file.

## [0.3.0] - 2026-03-29
- Added Alpine.js support inside `.antlers.html` files, including injected JavaScript intelligence and method navigation from Alpine expressions back to `x-data` methods.
- Improved Antlers editor support with Statamic-powered tag, modifier, and variable completion plus hover documentation sourced from the official docs.
- Added partial navigation, richer Antlers highlighting, and template-aware formatting improvements for mixed Antlers/HTML/Alpine files.
- Added `Tools > Statamic` actions for content query snippets, controller scaffolds, tag scaffolds, and modifier scaffolds in PhpStorm.
